# <a name="microsoft-365-guest-sharing-settings-reference"></a>Referencia de la configuración de uso compartido de invitados de Microsoft 365

En el artículo siguiente se proporciona una referencia para las distintas opciones de configuración que pueden afectar al uso compartido de usuarios externos a la organización en las cargas de trabajo de Microsoft 365: Teams, Grupos de Office 365, SharePoint y OneDrive. Esta configuración se encuentra en los centros de administración de Azure Active Directory, Microsoft 365, Teams y SharePoint.

- [Referencia de la configuración de uso compartido de invitados de Microsoft 365](https://docs.microsoft.com/microsoft-365/solutions/microsoft-365-guest-settings?view=o365-worldwide)
