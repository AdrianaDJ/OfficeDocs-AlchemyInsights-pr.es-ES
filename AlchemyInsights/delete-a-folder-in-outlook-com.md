# <a name="delete-a-folder"></a>Eliminar una carpeta

1. En el panel izquierdo, haga clic con el botón secundario en la carpeta o subcarpeta que quiera eliminar y seleccione **eliminar**. 
    
2. Seleccione **Aceptar** para confirmar. 
    
Para obtener más información, consulte [eliminar una carpeta en Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
