# <a name="access-denied-banned-sender"></a>Acceso denegado, no permitidas remitente

 **Error**: error de SMTP desde un servidor remoto para RCPT TO de comando, host: *xxxx* -com.mail.protection.outlook.com ( *xxx.xxx.xxx.xxx* ) motivo: 550 5.7.511 acceso denegado, prohibido remitente [ *xx.xxx.x.xxx* ]. 
  
Si este error se produce cuando se envía correo electrónico a un destinatario de Office 365, quitar la cuenta afectada de la lista de remitentes bloqueados en [https://sender.office.com](https://sender.office.com).
