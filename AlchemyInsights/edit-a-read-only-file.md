# <a name="edit-a-read-only-file"></a>Edición de un archivo de solo lectura

En las bibliotecas de documentos, al hacer clic en el nombre de un documento, el programa de Office puede abrirse de forma inesperada como un archivo de solo lectura. Para abrirla y editarla, haga clic en **abrir** (los tres puntos) y, a continuación, haga clic en **Editar.**
