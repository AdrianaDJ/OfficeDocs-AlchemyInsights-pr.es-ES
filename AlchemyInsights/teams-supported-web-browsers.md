# <a name="teams-supported-web-browsers"></a>Exploradores Web compatibles con Teams

Teams es totalmente compatible con los exploradores de Internet que aparecen en el artículo, [límites y especificaciones de Microsoft Teams](https://docs.microsoft.com/microsoftteams/limits-specifications-teams#browsers), con las excepciones indicadas para las llamadas y las reuniones.
