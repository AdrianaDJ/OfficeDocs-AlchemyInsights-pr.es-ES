# <a name="fix-onedrive-sync-problems"></a>Solucionar problemas de sincronización de OneDrive

La aplicación Asistente para soporte y recuperación puede ayudarle a identificar y corregir problemas comunes relacionados con la sincronización de archivos de OneDrive. 
  
[Descargue el Asistente para soporte y recuperación](https://aka.ms/sara), inicie sesión con su cuenta profesional o educativa y siga las instrucciones de la aplicación para solucionar problemas de OneDrive para la empresa. 
  
[Más pasos para la solución de problemas](https://go.microsoft.com/fwlink/?linkid=872097)
