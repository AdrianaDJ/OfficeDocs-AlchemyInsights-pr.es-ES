# <a name="web-only-limited-access"></a>Acceso limitado Web-Only

Web-Only acceso limitado es una variante del nivel de permisos "acceso limitado" que permite el acceso de los usuarios al objeto Web únicamente.

Para obtener más información, vea [niveles de permisos predeterminados](https://docs.microsoft.com/sharepoint/understanding-permission-levels#default-permission-levels).
