# <a name="learn-about-external-calendar-sharing-options"></a>Obtener información sobre las opciones de uso compartido de calendario externo

Si necesita programar reuniones con personas de fuera de la organización, Microsoft 365 permite que los calendarios estén disponibles para "usuarios externos", quienes necesitan ver el tiempo de disponibilidad pero no tienen cuentas de usuario para el entorno de Microsoft 365. Para obtener más información, vea [compartir calendarios con usuarios externos](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
