# <a name="change-a-users-name-and-email-address"></a>Cambiar la dirección de correo electrónico y de nombre de un usuario

Cambio de dirección de correo electrónico y de nombre de un usuario es fácil! Vaya a [Usuarios activos](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/users) y seleccione el usuario para editar su información. También puede usar este artículo para obtener instrucciones paso a paso: 
  
Cambiar una dirección de correo electrónico y un nombre de usuario en Office 365
  
 **Nota**: si cambia la dirección de correo electrónico principal del usuario, su dirección de correo electrónico antiguos seguirán siendo válido para recibir correo electrónico (Esto sucede automáticamente). Además, si la organización está en estado de híbrida, todos los cambios de nombre de usuario y de correo electrónico necesitan se realiza en Active Directory local y, a continuación, sincronizado en la nube.
