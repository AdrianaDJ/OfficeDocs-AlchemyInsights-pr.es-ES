# <a name="57750-client-blocked-from-sending-from-unregistered-domain"></a>5.7.750 cliente bloqueado el envío de dominio no registrada

El error se produce cuando un gran volumen de los mensajes se envían desde dominios que no se pueden aprovisionar a Office 365 (agregado como dominios aceptados y validado).
  
Para evitar este error, puede usar un conector de flujo de correo basado en el certificado que dominio del certificado es un dominio suministrado, o puede aprovisionar todos los dominios envío.
