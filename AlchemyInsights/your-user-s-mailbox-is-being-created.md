# <a name="user-mailbox-creation-is-taking-a-long-time"></a>La creación de buzones de usuario tarda mucho tiempo

Puede tardar hasta 24 horas antes de que un nuevo buzón esté listo para usarse.
  
Si el buzón del usuario se ha "preparando" durante más de una hora, pruebe a quitar la licencia de Exchange del usuario y volver a agregarla, o [póngase en contacto con el soporte técnico](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products?tabs=online).
