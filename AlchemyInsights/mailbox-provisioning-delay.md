# <a name="your-users-mailbox-is-being-created"></a>Se está creando el buzón de correo del usuario.

Puede tardar hasta una hora antes de que un nuevo buzón esté listo para usarse.
  
Si el buzón de un usuario se ha "preparando" durante más de una hora, puede intentar eliminarlo y, a continuación, volver a crearlo, o usar la opción "Permítanos llamar" para hablar con un ingeniero de soporte técnico.
