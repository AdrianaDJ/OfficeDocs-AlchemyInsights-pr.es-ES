# <a name="sharepoint-publishing-infrastructure"></a>Infraestructura de publicación de SharePoint


SharePoint incluye un conjunto clásico de características que permiten crear y publicar páginas web enriquecidas para la intranet. Active la infraestructura de publicación de SharePoint en el nivel de colección de sitios para activarla.

Para obtener más información, visite los siguientes vínculos.

[Habilitación de las características de publicación](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Características de publicación](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Notas:

- Las plantillas de sitio de publicación clásico incluyen las características de publicación de forma predeterminada.

- Las características de publicación clásicas no están disponibles en los sitios modernos. Los sitios modernos proporcionan una alternativa más nueva de forma predeterminada.
