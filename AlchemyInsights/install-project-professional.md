# <a name="install-project-professional"></a>Instalación de Project Professional

Vea [instalar Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281?ui=en-US&rs=en-US&ad=US) para obtener información sobre cómo instalar la versión de suscripción de Project online y la versión de no suscripción de Project 2019, 2016 o 2013. 

Para los proyectos 2010 o 2007, siga los pasos descritos en [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) u [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Project se vende como una aplicación independiente, por lo que, si aún no la tiene, [compre o pruebe Project](https://products.office.com/project).
