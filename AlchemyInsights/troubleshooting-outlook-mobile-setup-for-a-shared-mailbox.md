# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Solución de problemas de configuración de Outlook Mobile para un buzón compartido

No puede acceder al buzón compartido desde la aplicación de Outlook instalada en el dispositivo móvil. ¿Quiere esta característica? Que el equipo de ingeniería de Outlook lo sepa. [Vote aquí](https://go.microsoft.com/fwlink/?linked=862116).
  
Solo puede usar un buzón compartido en un dispositivo móvil si abre el explorador, inicia sesión y, a continuación, va a Outlook en la Web. Para obtener instrucciones, vea [Agregar un buzón compartido en Outlook en la web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Desde Outlook en la Web, podrá acceder al buzón compartido.
  
Obtenga más información sobre cómo [obtener ayuda en la aplicación para Outlook para iOS y Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
