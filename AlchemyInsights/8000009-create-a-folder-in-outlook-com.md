# <a name="create-a-folder"></a>Cree una carpeta

1. En el panel izquierdo, seleccione **Crear nueva carpeta**![crear el nuevo botón de la carpeta](media/d8e28612-fbdb-4d28-a4d0-14f7834cfd97.png) junto a **las carpetas**. 
    
2. En el cuadro nueva carpeta, escriba un nombre para la carpeta y presione ENTRAR.
    
Para obtener más información, consulte [crear una carpeta en Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
