# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Utilizar a Marta para diagnosticar y resolver problemas de autenticación de Outlook

Se recomienda que utilice el [Outlook pide constantemente la contraseña de](https://aka.ms/SaRA-OutlookPwdPrompt) diagnóstico en el equipo afectado para solucionar los problemas donde Outlook continuamente solicita una contraseña. Este diagnóstico [Marta](https://diagnostics.office.com/#/) realiza comprobaciones automatizadas y devuelve las posibles soluciones que puede utilizar para solucionar los problemas detectados.
