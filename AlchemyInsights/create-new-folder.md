# <a name="create-a-new-folder-in-outlookcom"></a>Cree una carpeta nueva en Outlook.com

1. En **carpetas** en el panel de navegación, seleccione **nueva carpeta**.

2. Escriba un nombre para la carpeta y presione Entrar.

Para obtener más información, consulte [crear una carpeta en Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
