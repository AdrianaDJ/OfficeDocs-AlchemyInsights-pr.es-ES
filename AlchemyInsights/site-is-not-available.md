# <a name="site-is-not-available"></a>El sitio no está disponible

Si el sitio de SharePoint o de OneDrive no está disponible para varios usuarios, es posible que haya un problema de servicio temporal. Compruebe el [Panel de estado del servicio](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver si el inquilino se ve afectado.
