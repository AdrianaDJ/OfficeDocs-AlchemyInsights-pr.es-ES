# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Impedir la descarga de archivos de SharePoint o OneDrive

Hay varios métodos que se pueden usar para impedir o bloquear las descargas. Visite los siguientes vínculos para los métodos respectivos:

- [Bloquear la descarga de archivos con el acceso condicional de Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Control de acceso desde dispositivos no administrados](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Configurar directivas de IRM en bibliotecas de documentos de SharePoint para limitar la descarga de contenido](https://docs.microsoft.com/microsoft-365/compliance/set-up-irm-in-sp-admin-center)
