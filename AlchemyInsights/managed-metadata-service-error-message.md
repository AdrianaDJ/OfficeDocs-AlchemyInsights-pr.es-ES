# <a name="managed-metadata-service-error-message"></a>Mensaje de error de Metadata Service administrado

Cuando se abre el almacén de términos, es posible que vea un mensaje de error: "el Managed Metadata Service o la conexión no está disponible. Puede que no se haya iniciado el grupo de aplicaciones o el servicio Web de metadatos administrados. Póngase en contacto con el administrador."
  
Si han transcurrido más de 24 horas dado que primero se han visto este problema, inicie sesión un caso de soporte técnico. Sabemos que 24 horas pueda parecer mucho tiempo. En muchos casos, ya trabajamos en una solución así háganos al menos 24 horas.
