# <a name="payment-methods-for-office-365-for-business"></a>Métodos de pago para Office 365 para empresas

Cuando compre una suscripción de Office 365 para empresas directamente desde Microsoft, podrá pagar con una tarjeta de crédito o débito. En algunos países o regiones, también puede pagar con una cuenta bancaria.
  
Si elige usar una tarjeta de crédito como método de pago, puede obtener información sobre cómo administrar y solucionar problemas desde [Agregar, actualizar o quitar una tarjeta de crédito o cuenta bancaria en Office 365 para empresas](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/add-update-or-remove-credit-card-or-bank-account).
  
En este artículo también se incluyen sugerencias para solucionar problemas de Cuándo se rechaza su tarjeta de crédito e información sobre cómo cambiar la información de las tarjetas de crédito para que la tarjeta se acepte.
