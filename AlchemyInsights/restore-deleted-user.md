# <a name="restore-a-user-in-office-365"></a>Restaurar un usuario en Office 365

Restauración de un usuario en Office 365 mediante estos pasos:
  
1. Vaya a [usuarios \> eliminan los usuarios](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/deletedusers).
    
2. Elija el usuario, a continuación, seleccione **Restaurar**.
    
3. Siga las indicaciones para configurar la contraseña.
    
4. Haga clic en **Enviar correo electrónico y cerrar**, y que hayan acabado!
    
¿No era fácil? Para obtener más detalles y pasos con capturas de pantalla, consulte este artículo: [restauración de un usuario en Office 365](https://support.office.com/article/https://support.office.com/en-us/article/Restore-a-user-in-Office-365-2c261e42-5dd1-48b0-845f-2a016d29cfc1.aspx)
