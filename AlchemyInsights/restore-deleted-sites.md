# <a name="restore-deleted-sites"></a>Restaurar sitios eliminados

Los sitios eliminados se conservan durante 93 días. Después de 93 días, los sitios y todo su contenido y configuración se eliminan de forma permanente, incluidas las listas, las bibliotecas, las páginas y los subsitios.

- Para obtener más información, consulte [Restaurar sitios eliminados](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection).
