# <a name="move-a-folder"></a>Mover una carpeta

Puede arrastrar carpetas bajo **Favoritos** para reordenarlos.
  
No puede reordenar carpetas en **carpetas** o mover una subcarpeta para convertirla en una carpeta.

*Nota:* Si mueve una carpeta para convertirla en una subcarpeta, cree una carpeta nueva y copie todo el correo electrónico de la subcarpeta a la nueva carpeta.
  
Para obtener más información, consulte [trabajar con carpetas de mensajes en Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a).
