# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnosticar y resolver problemas de conectividad de Outlook

Para diagnosticar y solucionar los problemas de conectividad de Outlook, puede ejecutar la [herramienta de conectividad de Outlook](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) en el equipo afectado.
  
Esta herramienta realizará comprobaciones de diagnóstico y sugerirá posibles soluciones para solucionar el problema de conectividad de Outlook.
