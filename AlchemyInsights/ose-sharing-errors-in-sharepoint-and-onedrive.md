# <a name="ose-sharing-errors"></a>Errores de uso compartido de OSE

En este artículo se tratan los distintos errores que pueden surgir al compartir archivos o carpetas de SharePoint Online o OneDrive para la empresa en Microsoft 365. Debe ser un administrador global o de SharePoint en su organización para solucionar estos errores. Si no tiene el rol de administrador, póngase en contacto con el soporte técnico e indique el código de error.

Para obtener más información sobre los códigos de error de OSE, consulte [compartir errores en SharePoint y OneDrive](https://docs.microsoft.com/sharepoint/sharepoint-onedrive-error-message).
