# <a name="a-user-has-two-mailboxes"></a>Un usuario tiene dos buzones de correo

Los entornos híbridos que usan Azure Active Directory Connect (AAD Connect) o dirSync podrían causar por accidente que un usuario tenga dos buzones de correo: uno en la organización local y otro en la nube.

Para resolver este problema, consulte [borrar permanentemente la información del buzón anterior](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
