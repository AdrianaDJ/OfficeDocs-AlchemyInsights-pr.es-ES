# <a name="create-a-shared-mailbox"></a>Creación de un buzón compartido

Un buzón de correo compartido permite a más de una persona enviar y recibir correo electrónico con un alias, como sales@contoso.com, sin asignar otra licencia. Para crear un buzón compartido, vaya a la [Guía de correo electrónico compartido](https://portal.office.com/adminportal/home).
  
Vea el artículo [crear un buzón compartido](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) para obtener sugerencias y más información sobre el uso de un buzón compartido.
