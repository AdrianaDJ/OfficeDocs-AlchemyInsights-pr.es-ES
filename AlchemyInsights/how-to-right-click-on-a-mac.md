# <a name="how-to-right-click-on-a-mac"></a>Cómo hacer clic con el botón secundario en un equipo Mac

En equipos Mac, el clic con el botón secundario se conoce como clic o control secundario. Si el mouse (ratón), el control de vista o cualquier otro dispositivo de entrada no incluye un botón de clic con el botón secundario u otro método para realizar un clic con el botón secundario, mantén presionada la **tecla control** en el teclado mientras hace clic. 
  
Para obtener más información, vaya a [soporte técnico de Apple](https://go.microsoft.com/fwlink/?linkid=2022220&amp;clcid=0x409).
