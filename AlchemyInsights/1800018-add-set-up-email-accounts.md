# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>Problemas de configuración de una cuenta de correo electrónico de Exchange en Outlook

Si tiene problemas de configuración de una cuenta de Exchange en Outlook, puede ejecutar la [herramienta de configuración de cuenta de Outlook](https://aka.ms/SaRA-OutlookSetupProfile) en el equipo afectado. Que llevar a cabo comprobaciones diversas y proporcionará las posibles soluciones para el problema de configuración de cuenta. 
  
Si sigue teniendo problemas, vea [que Outlook no se puede configurar un nuevo perfil mediante el uso de detección automática de Exchange para un buzón de Exchange Online en Office 365](https://support.microsoft.com/help/2404385/outlook-can-t-set-up-a-new-profile-by-using-exchange-autodiscover-for).
