# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Usar SaRA para diagnosticar y resolver problemas de autenticación de Outlook

Se recomienda encarecidamente usar el diagnóstico de [Outlook para](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) que el diagnóstico de contraseñas del equipo afectado solucione problemas en los que Outlook le pida continuamente una contraseña. Este diagnóstico de [SaRA](https://diagnostics.office.com/#/) realiza comprobaciones automatizadas y devuelve las posibles soluciones que puede usar para solucionar los problemas detectados.
