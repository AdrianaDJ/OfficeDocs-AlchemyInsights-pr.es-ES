# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>¿Obtención de un error de permiso al abrir un buzón compartido en Outlook en el web?

Si se produce el error **se produjo algún error. No tiene permiso para abrir este buzón de correo.** Cuando intenta abrir un buzón compartido en Outlook en el web, su cuenta no es posible que tiene permiso para acceder al buzón compartido. El Administrador de la organización puede asignar los permisos de cuenta para el buzón compartido siguiendo las instrucciones que aparecen en el tema [crear un buzón compartido](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9).
