# <a name="setting-up-a-hybrid-environment"></a>Configuración de un entorno híbrido

Para que los usuarios puedan trabajar sin problemas sin importar si sus datos se almacenan en servidores locales o en Office 365, es posible que deba configurar un entorno híbrido. Para obtener información general sobre las opciones, consulte [Office 365 Integration with on-](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx)premises environments.
