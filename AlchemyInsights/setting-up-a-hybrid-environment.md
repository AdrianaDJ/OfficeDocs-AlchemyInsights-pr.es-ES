# <a name="setting-up-a-hybrid-environment"></a>Configuración de un entorno híbrido

Para que los usuarios puedan trabajar sin problemas sin importar si sus datos se almacenan en servidores locales o en Microsoft 365, es posible que deba configurar un entorno híbrido. Para obtener información general sobre las opciones, consulte [Microsoft 365 Integration with on-premises Environments](https://docs.microsoft.com/office365/enterprise/office-365-integration).
