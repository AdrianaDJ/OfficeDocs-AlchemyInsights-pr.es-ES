# <a name="removing-the-last-on-premises-exchange-server"></a>Quitar el último servidor de Exchange local

Antes de quitar el último servidor de Exchange de la organización local, debe asegurarse de que comprende las implicaciones de esta acción para ayudarle a decidir si necesita ese último servidor para cualquier otro fin. Para obtener más información, consulte [how and When to recommission Your on-premises Exchange Servers in a Hybrid Deployment](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
