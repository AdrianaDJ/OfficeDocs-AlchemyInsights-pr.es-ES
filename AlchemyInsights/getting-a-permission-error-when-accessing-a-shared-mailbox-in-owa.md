# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Obtener un error de permisos al abrir un buzón compartido en Outlook en la web

Si recibe un error, se produjo **un error. No tiene permiso para abrir este buzón.** cuando intenta abrir un buzón compartido en Outlook en la web, es posible que su cuenta no tenga permiso para obtener acceso al buzón compartido. El administrador de la organización puede asignar los permisos de cuenta al buzón compartido siguiendo las instrucciones que se indican en el tema [crear un buzón compartido](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox).
