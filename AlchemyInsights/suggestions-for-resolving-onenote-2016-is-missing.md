# <a name="suggestions-for-resolving-onenote-2016-is-missing"></a>Faltan sugerencias para resolver OneNote 2016

Después de instalar Microsoft Office 2019 u Office 365 en un equipo que ejecuta cualquier edición de Windows 10, es posible que OneNote 2016 no aparezca en la lista de aplicaciones de escritorio instaladas.

De forma predeterminada, Office 2019 y Office 365 no instalarán automáticamente la versión de escritorio de OneNote 2016, pero si sigue necesitando o prefiere usar esta versión, puede descargarla por separado y usarla junto con las nuevas aplicaciones de Office 2019.

Siga [estos pasos](https://support.office.com/article/OneNote-2016-is-missing-after-installing-Office-2019-or-Office-365-1844ba87-7248-4bd8-a735-66a52f98e6e5) para obtener más información sobre cómo descargar la edición de 32 bits o de 64 bits de OneNote 2016 para Windows.
