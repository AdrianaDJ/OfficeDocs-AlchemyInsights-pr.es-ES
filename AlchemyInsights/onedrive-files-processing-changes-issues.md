# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: solucionar problemas "procesando cambios"

Si ve "procesando cambios" durante un período de tiempo prolongado, puede ser debido a que tiene abierto un archivo en línea, que hay muchos archivos bloqueados en una cola o una sincronización de archivos de gran tamaño, vea [OneDrive bloqueado en el procesamiento](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Lea este artículo para solucionar los problemas de "procesamiento de cambios". También verá este mensaje si tiene un archivo que cambia todo el tiempo (por ejemplo, Outlook. PST o una base de datos). Obtenga información sobre cómo [mover los archivos de datos de Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) que está usando con frecuencia fuera de OneDrive.
