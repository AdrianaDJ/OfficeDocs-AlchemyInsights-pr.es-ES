# <a name="changes-to-external-user-access"></a>Cambios realizados en el acceso de usuarios externos

Iniciar el 23 de marzo de 2018, los usuarios externos ya no podrán tener acceso al contenido que se ha compartido con todos los usuarios, todos los usuarios autenticados o todos los usuarios de formularios, incluso si han sido invitados o agregado a grupos autorizados. Los usuarios externos solo podrán tener acceso al contenido que se ha compartido con el usuario o grupos a los que pertenece el usuario. Si desea conservar la configuración existente, consulte:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
