# <a name="create-and-use-a-shared-mailbox"></a>Crear y usar un buzón compartido

Un buzón de correo compartido permite a más de una persona enviar y recibir correo electrónico con un alias, como sales@contoso.com, sin asignar otra licencia.
  
- [Ve aquí para empezar](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)
    
También puede leer el artículo [crear un buzón compartido](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx) que le indicará todo lo que necesita saber sobre los buzones compartidos, incluido cómo usarlos.
