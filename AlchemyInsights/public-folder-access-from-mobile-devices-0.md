# <a name="public-folder-access-from-mobile-devices"></a>Acceso a carpetas públicas desde dispositivos móviles

Las carpetas públicas no son accesibles desde dispositivos móviles.
  
Si desea que tienen funcionalidad similar a las carpetas públicas y obtener acceso a él desde dispositivos móviles, vea [Aprenda acerca de los grupos de Office 365](https://support.office.com/en-us/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) para una mejor alternativa.
