# <a name="duplicate-object-synchronization-errors"></a>Errores de sincronización del objeto duplicado

Si recibe uno de los siguientes errores durante una sincronización:
  
- Un objeto sincronizado con la misma dirección de proxy ya existe en el directorio de Microsoft Online Services.
    
- Un mensaje similar para UserPrincipalName.
    
Utilice el nuevo estado de AADConnect diagnóstico para resolver los errores de objeto duplicado. Para obtener más información, vea [diagnosticar y corregir los errores de sincronización de atributo duplicada](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
