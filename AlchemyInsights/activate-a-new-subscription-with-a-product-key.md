# <a name="activate-a-new-subscription-with-a-product-key"></a>Activar una nueva suscripción con una clave de producto

1. Para activar una nueva suscripción con la clave de producto, vaya a la [https://office.com/setupkey](https://office.com/setupkey) página. 
    
2. Escriba la clave de producto de Office 365 y siga las indicaciones para activar su suscripción. Si tiene problemas con la activación, consulte esta [lista de errores de clave de producto comunes y soluciones](https://support.office.com/article/88d337ab-e7b1-43eb-a25e-7d6204e91099).
