# <a name="install-office"></a>Instalación de Office

Para instalar Office, los usuarios deben tener un [producto de negocio de Office 365](https://support.office.com/article/f8ab5e25-bf3f-4a47-b264-174b1ee925fd?wt.mc_id=Alchemy_ClientDIA) que incluye las aplicaciones de escritorio y un [asignado licencia](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users).
  
A continuación, se puede instalar Office los pasos siguientes:
  
1. Vaya a [portal.office.com/OLS/MySoftware.aspx](https://portal.office.com/OLS/MySoftware.aspx)
    
2. En la página principal de Office, seleccione el idioma que desee y, a continuación, seleccione Opciones avanzadas para elegir entre la versión de 32 bits o 64 bits
    
3. Seleccione **instalar Office**.
    
Para obtener más información, vea [descargar e instalar o volver a instalar Office en un PC o Mac](https://support.office.com/article/4414eaaf-0478-48be-9c42-23adc4716658?wt.mc_id=Alchemy_ClientDIA)
