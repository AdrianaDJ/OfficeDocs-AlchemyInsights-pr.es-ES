# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Solución de problemas de escenarios de actualización-errores de mosaico

Si experimenta errores de actualización del icono en el servicio de Power BI, espere unos 10-15 minutos para que la actualización se refleje en los mosaicos del panel. Si aún no aparece, vuelva a anclar la visualización en el panel.

Para obtener más información: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
