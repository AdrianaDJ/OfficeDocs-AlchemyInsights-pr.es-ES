# <a name="whats-a-hub-site"></a>¿Qué es un sitio concentrador?

Un sitio concentrador le ayudará a organizar los sitios y les proporcionará una navegación y personalización de marca comunes. Los administradores globales o de SharePoint podrán crear sitios de concentradores y especificar qué usuarios pueden asociar sus sitios con el sitio concentrador. Para obtener más información, consulte [sitios concentradores de SharePoint](https://go.microsoft.com/fwlink/?linkid=869388).
