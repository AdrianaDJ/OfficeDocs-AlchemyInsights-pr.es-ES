# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>Problemas de disponibilidad de calendario en entornos híbridos

Si los usuarios tienen problemas vean calendario libre/ocupado de la información en entornos híbridos, vuelva a ejecutar al Asistente de configuración híbrida. Para obtener más información acerca del Asistente para la configuración híbrida, vea [Asistente para la configuración híbrida](https://go.microsoft.com/fwlink/p/?linkid=528149).
  
Para solucionar problemas de los pasos en entornos híbridos, vea [Troubleshoot una implementación híbrida](https://technet.microsoft.com/library/jj659053.aspx).
