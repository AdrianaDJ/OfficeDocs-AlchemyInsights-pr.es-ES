# <a name="how-to-select-all-in-outlookcom"></a>Cómo "seleccionar todo" en Outlook.com

Solo puede seleccionar todo si todos los mensajes están visibles. Esto es para evitar acciones accidentales en los mensajes que no puede ver.

**Ctrl + clic** selecciona varios mensajes individuales.

**Mayús + clic** selecciona todos los mensajes entre el primer y último mensaje seleccionado.
