# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Error de Outlook: "No se puede iniciar Microsoft Outlook"

¿Tiene problemas al iniciar Outlook 2016, Outlook 2013 o Outlook 2010 o recibir "no puede iniciar Microsoft Outlook. ¿No se puede abrir la ventana de Outlook"mensajes de error?
  
En primer lugar, use los pasos de solución de problemas de [este artículo](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) para ayudar a resolver estos problemas de inicio. 
  
También puede ejecutar la [herramienta de diagnóstico avanzada](https://aka.ms/SaRA-OutlookAdvDiagnostics) en el equipo afectado.
