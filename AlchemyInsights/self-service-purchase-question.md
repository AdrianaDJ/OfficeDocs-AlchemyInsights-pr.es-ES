# <a name="self-service-purchase"></a>Compra de servicio automático

**¿Qué funcionalidades tiene un administrador para las compras de autoservicio?**

Los administradores pueden ver todas las compras de autoservicio realizadas en su organización en el centro de administración de Microsoft 365. Pueden ver el producto, el nombre del comprador, las suscripciones adquiridas, la fecha de expiración, el historial de pedidos, el precio de compra y los usuarios asignados para cada una de las compras de servicios automáticos.  Si es necesario para su organización, los administradores podrán desactivar la compra de autoservicio para cada producto mediante PowerShell.  Los administradores tienen las mismas directivas de administración y acceso de datos que los productos adquiridos a través de la compra de autoservicio o de forma centralizada.
