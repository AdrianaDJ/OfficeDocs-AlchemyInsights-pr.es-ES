# <a name="shared-mailbox-or-calendar-issue-in-outlook-for-mac"></a>¿Problema de buzón de correo o calendario compartido en Outlook para Mac?

Podemos ayudarle con mayor rapidez si se en contacto con nosotros directamente desde la aplicación. **Vaya a Mac del usuario afectado, abra Outlook y haga clic en Ayuda \> compatibilidad con contactos**. 
  
Para obtener más información acerca del uso de Outlook de soporte de aplicación de Mac, vea este artículo: [característica de contactar con el nuevo servicio de 2016 de Outlook para Mac](https://support.office.com/article/https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
