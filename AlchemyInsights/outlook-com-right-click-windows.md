# <a name="how-to-right-click-on-a-tablet"></a>Cómo hacer clic con el botón secundario en una tableta

Para hacer clic con el botón secundario en una superficie o en una tableta de Windows, pulse y mantenga presionada la selección durante unos segundos. Cuando libere, aparecerá el menú contextual.
