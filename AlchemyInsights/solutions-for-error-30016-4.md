# <a name="solutions-for-error-30016-4"></a>Soluciones para el error 30016-4

Cuando intenta desinstalar Office 365 ProPlus desde un equipo, es posible que reciba el siguiente mensaje de error:
  
> "Algo salió mal 30016-4"
    
Este problema se produce si la unidad **% temp %** está asignada a una unidad que no sea **% archivos de programa %**. 
  
Para corregir, quitar completamente Office desde el equipo.
  
1. Descargue e instale el [Ayudante para la recuperación y soporte técnico](https://aka.ms/SARA-OfficeUninstall-Alchemy).
    
2. Cuando se abre, en **productos de desinstalación de Office** , seleccione la versión que desea desinstalar y seleccione **siguiente**. 
    
3. Continuar con el proceso a través de pantallas y reinicie el equipo cuando se le solicite.
    
    Después de reiniciar, el soporte técnico y el Asistente de recuperación para volver a se abre automáticamente para completar el proceso de desinstalación.
    
4. Siga las indicaciones restantes y volver a instalar Office o cerrar al asistente.
