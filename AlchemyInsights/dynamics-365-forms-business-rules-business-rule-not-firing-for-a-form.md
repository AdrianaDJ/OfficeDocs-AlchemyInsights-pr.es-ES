# <a name="business-rule-not-firing-for-a-form"></a>No se ha desencadenado la regla de negocio para un formulario

Es posible que una regla de negocio no se ejecute porque el campo al que se hace referencia en la regla de negocio no se incluye con el formulario. Para obtener más información, vea [¿la regla de negocio no se ha desencadenado para un formulario?](https://docs.microsoft.com/powerapps/maker/model-driven-apps/create-business-rules-recommendations-apply-logic-form#is-your-business-rule-not-firing-for-a-form).
