# <a name="how-to-share-sharepoint-files-or-folders"></a>Cómo compartir archivos o carpetas de SharePoint

Los archivos que se almacenan en un sitio de SharePoint suelen estar disponibles para todos los usuarios con permisos para el sitio, pero es posible que desee compartir archivos o carpetas específicos con personas que, de otro modo, tienen acceso al sitio. Al compartir archivos y carpetas, puede decidir si desea que los usuarios los editen o simplemente los vean.
