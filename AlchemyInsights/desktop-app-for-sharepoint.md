# <a name="desktop-app-for-sharepoint"></a>Aplicación de escritorio para SharePoint

Su organización se especifica en el programa de la versión preliminar privada para la aplicación de escritorio de SharePoint. Actualmente, Microsoft no es compatible con la aplicación de escritorio de SharePoint (versión preliminar privada). Use sus partners de vista previa de Microsoft Private listados para obtener ayuda con la aplicación de escritorio de SharePoint.
