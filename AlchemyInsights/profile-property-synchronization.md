# <a name="profile-property-synchronization"></a>Sincronización de propiedades de perfil

No es posible asignar propiedades adicionales para que la importación de AD se sincronice desde Azure Active Directory con la aplicación de Perfil de usuario. La importación de AD está limitada a un conjunto de propiedades preconfiguradas para garantizar un rendimiento coherente del trabajo del temporizador.
  
[Obtenga más información sobre las propiedades de perfil que se sincronizan](https://go.microsoft.com/fwlink/?linkid=875671)
