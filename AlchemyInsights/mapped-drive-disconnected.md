# <a name="sharepoint-libraries-mapped-to-network-drives-become-disconnected"></a>Las bibliotecas de SharePoint asignadas a unidades de red se desconectan

La asignación de una biblioteca de documentos de SharePoint a una unidad de red es temporal. En ocasiones, debe abrir el sitio de SharePoint en Internet Explorer y seleccionar **permanecer con sesión iniciada** para evitar que la sesión expire. 
  
[Obtenga ayuda para configurar y solucionar problemas de unidades de red asignadas](https://support.office.com/article/ef399c67-4578-4c3a-adbe-0b489084eabe.aspx).
  
Para trabajar con una biblioteca en el explorador de archivos, [sincronice los archivos de SharePoint con el nuevo cliente de sincronización de OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) , que incluye [los archivos a petición](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Obtenga acceso a todos los archivos en OneDrive sin usar el espacio de almacenamiento local.
