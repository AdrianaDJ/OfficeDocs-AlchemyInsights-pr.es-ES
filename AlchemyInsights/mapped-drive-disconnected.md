# <a name="sharepoint-libraries-mapped-to-network-drives-become-disconnected"></a>Las bibliotecas de SharePoint que se asignan a las unidades de red se desconectan

Asignación de una biblioteca de documentos de SharePoint a una unidad de red es temporal. Ocasionalmente, debe abrir el sitio de SharePoint en el Explorador de Internet y seleccione **mantener la sesión de** para evitar que la sesión caduca. 
  
[Obtener ayuda para configurar y solucionar problemas de unidades de red asignadas](https://support.office.com/article/ef399c67-4578-4c3a-adbe-0b489084eabe.aspx).
  
Para que funcione con una biblioteca en el Explorador de archivos, [sincronizar los archivos de SharePoint con el nuevo cliente de sincronización de OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) que incluye [Los archivos a petición](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Tener acceso a todos los archivos en OneDrive sin usar el espacio de almacenamiento local.
