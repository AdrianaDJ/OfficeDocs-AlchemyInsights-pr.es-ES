# <a name="configure-multi-factor-authentication"></a>Configurar la autenticación multifactor

Para configurar la MFA, consulte [set up multi-factor Authentication](https://docs.microsoft.com/office365/admin/security-and-compliance/set-up-multi-factor-authentication).

Tenga en cuenta que debe ser un administrador global para habilitar y configurar la MFA.
