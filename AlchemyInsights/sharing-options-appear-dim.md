# <a name="sharing-options-appear-dim-or-are-not-available"></a>Las opciones de uso compartido aparecen atenuadas o no están disponibles

Si algunas de las opciones de uso compartido o no están disponibles al intentar compartir desde SharePoint Online o OneDrive, vea [las opciones de uso compartido se muestran atenuadas al compartir desde SharePoint Online o onedrive](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive).
