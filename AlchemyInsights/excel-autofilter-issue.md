# <a name="excel-autofilter-issue"></a>Problema de Autofiltro de Excel

Al usar Excel 2016, si la lista desplegable Autofiltro no muestra todas las casillas de verificación disponibles, o las casillas de verificación no se puede hacer clic, puede deberse a un problema conocido en versiones anteriores de Excel. 
  
Ahora, este problema se corrige en **versión 1707 (8326.2073)** o posterior. Para obtener la actualización inmediatamente, abra Excel y elija **archivo** \> **cuenta** \> **Opciones de actualización de** \> **Actualizar ahora**.
