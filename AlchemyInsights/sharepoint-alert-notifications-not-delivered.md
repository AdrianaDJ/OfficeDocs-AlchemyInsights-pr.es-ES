# <a name="sharepoint-alert-notifications-not-delivered"></a>Notificaciones de alerta de SharePoint no entregadas 

<p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Determine si no se entregan <strong>todas las alertas</strong> o si no se entrega una <strong>alerta individual</strong> de un archivo o biblioteca específicos.</span></p> <ol> <li><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Si no se entregan <strong style="mso-bidi-font-weight: normal;">todas las alertas</strong> de varios archivos o bibliotecas, visite el <a href="https://admin.microsoft.com/AdminPortal/Home#/servicehealth">Panel de estado del servicio</a> para comprobar si hay algún asesor o incidente que pueda estar ocurriendo con SharePoint o Exchange. El problema podría estar relacionado con la capacidad de alerta de SharePoint o con retrasos en los correos electrónicos a través de Exchange. También será importante tener en cuenta si se va a entregar otro correo electrónico y, de no ser así, es probable que el problema se produzca con retrasos de Exchange. <br /><br /></span></li> <li style="line-height: normal; font-size: 11pt; font-style: normal; font-weight: 400;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Si un</span> <strong><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">alerta individual</span></strong> <span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">de un archivo o biblioteca específicos no se entrega, puede intentar eliminarlo y volver a crearlo. Consulte</span> <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> <a href="https://support.office.com/en-us/article/manage-view-or-delete-sharepoint-alerts-99dfb19c-9a90-4a8c-aba1-aa8c8afb0de2#ID0EAADAAA=Online">administrar, ver o eliminar alertas de SharePoint</a> para volver a crear la alerta.</span></li> </ol> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Para obtener más información sobre cómo configurar alertas, vea <span style="color: #333333;"> <a href="https://support.office.com/en-us/article/create-an-alert-to-get-notified-when-a-file-or-folder-changes-in-sharepoint-e5a79e7b-a146-46da-a9ef-d65409ba8918">crear una alerta para recibir una notificación cuando cambia un archivo o una carpeta en SharePoint</a>.</span></span></p> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Para obtener más información sobre la solución de problemas de alertas, consulte <a href="https://support.office.com/en-us/article/users-don-t-receive-sharepoint-online-alert-notifications-14fc22dd-e873-482c-844d-f67ad41313f1">&rsquo;usuarios no recibir notificaciones de alerta de SharePoint Online</a>.</span></p> <p>&nbsp;</p>
