# <a name="classification-labels"></a>Etiquetas de clasificación

Con las etiquetas de confidencialidad o clasificación, puede clasificar y ayudar a proteger el contenido confidencial, al tiempo que se asegura de que la productividad y la capacidad de colaboración de sus contactos no se vean obstaculizadas.

Para obtener más información, consulte el siguiente [artículo](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels).
