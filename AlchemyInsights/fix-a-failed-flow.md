# <a name="fix-a-flow-that-failed"></a>Corregir un flujo con errores

1. Vaya a [Microsoft Flow](https://flow.microsoft.com/)y haga clic en el icono de notificaciones en la esquina superior derecha. (O en la aplicación móvil, seleccione la pestaña actividad). Seleccione el flujo que no ha podido ver los detalles.
    
2. Busque el paso con el icono de exclamación roja para ver el mensaje de error del flujo.
    
3. Según el mensaje de error, podrá editar el flujo y corregirlo. 
    
[Errores de flujo comunes](https://go.microsoft.com/fwlink/?linkid=872110)
