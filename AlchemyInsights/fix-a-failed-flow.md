# <a name="fix-a-flow-that-failed"></a>Corregir un flujo con errores

1. Vaya a [Flujo de Microsoft](https://flow.microsoft.com/)y haga clic en el icono de notificaciones en la parte superior derecha. (O en la aplicación móvil, seleccione la ficha actividad). Seleccione el flujo de que no se pudo ver los detalles.
    
2. Busque el paso con el icono de signo de exclamación rojo para ver el mensaje de error para el flujo.
    
3. Según el mensaje de error, debe ser capaz de modificar el flujo y corregirlo. 
    
[Errores comunes de flujo](https://go.microsoft.com/fwlink/?linkid=872110)
