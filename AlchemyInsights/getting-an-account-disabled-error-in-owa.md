# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>¿Se obtiene un error de cuenta deshabilitada en Outlook en la web?

Si recibe el error con `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` el que se ha **deshabilitado la cuenta** cuando intenta abrir Outlook en la web (anteriormente conocido como Outlook Web App), es posible que el administrador de la organización haya deshabilitado el acceso a Outlook en la Web. El administrador puede solucionar este error si sigue los pasos descritos en el tema [enable or Disable Outlook Web App for a Mailbox](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
