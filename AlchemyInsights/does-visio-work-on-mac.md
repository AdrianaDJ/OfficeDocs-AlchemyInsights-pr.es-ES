# <a name="does-visio-work-on-a-mac"></a>¿Funciona Visio en un equipo Mac?

No hay ninguna aplicación de escritorio de Visio para Mac. Sin embargo, siempre que el usuario que necesite la aplicación tenga [asignada una licencia de Visio online](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799), puede [ver, crear y editar diagramas en Visio online](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) en un explorador Web.
