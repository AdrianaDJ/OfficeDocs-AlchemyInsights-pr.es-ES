# <a name="sharepoint-online-throttling"></a>SharePoint Online limitación

SharePoint Online utiliza limitación para mantener un rendimiento óptimo y la confiabilidad del servicio de SharePoint Online. Superado la limitación el número de acciones de usuario o usuarios simultáneos llamadas (por secuencia de comandos o código) para evitar el uso excesivo de los recursos. Si limita a obtener, 99% de tiempo es debido a código personalizado.
