# <a name="sharepoint-online-throttling"></a>Limitación de SharePoint Online

SharePoint Online utiliza limitación para mantener un rendimiento óptimo y la confiabilidad del servicio SharePoint Online. La limitación se llama el número de acciones de usuario o simultáneos (por secuencia de comandos o código) para evitar el uso excesivo de los recursos. Si limita a obtener, 99% de tiempo es debido a código personalizado.
  
Para obtener más información sobre la limitación, vea [evitar limitaciones o bloqueos en SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
