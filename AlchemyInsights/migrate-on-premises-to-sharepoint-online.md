# <a name="migrate-on-premises-to-sharepoint-online"></a>Migrar de forma local a SharePoint Online

Para migrar de una granja de servidores de SharePoint local a SharePoint Online, puede usar la herramienta de migración de SharePoint. Para obtener más información acerca de la herramienta, consulte [migrar a SharePoint Online.](https://go.microsoft.com/fwlink/?linkid=2019574)
