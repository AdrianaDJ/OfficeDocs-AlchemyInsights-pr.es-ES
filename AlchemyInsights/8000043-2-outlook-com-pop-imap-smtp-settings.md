# <a name="pop-imap-and-smtp-settings-for-outlookcom"></a>Configuración de POP, IMAP y SMTP para Outlook.com

Si desea agregar su cuenta de Outlook.com a otra aplicación de correo que admita POP o IMAP, esta es la configuración manual del servidor que necesitará:

- **Nombre del servidor IMAP**: IMAP-mail.Outlook.com
- **Puerto IMAP**: 993
- **Nombre del servidor pop**: pop-mail.Outlook.com
- **Puerto pop**: 995
- **Método de cifraDo pop**: TLS
- **Nombre del servidor SMTP**: SMTP-mail.Outlook.com
- **Puerto SMTP**: 587
- **Método de cifraDo SMTP**: STARTTLS

Obtenga más información sobre la [configuración de pop, IMAP y SMTP para Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001402&clcid=0x409).
