# <a name="spam-filter-not-working"></a>El filtro de correo no deseado no funciona

Si cree que ha recibido un mensaje de suplantación de identidad (phishing) o suplantación de identidad, marque el mensaje como correo no deseado y, después, vaya a **correo no deseado**, seleccione el mensaje y seleccione **no correo no deseado** \> **** para marcarlo como una estafa de suplantación de identidad. Para obtener más información acerca del abuso, el phishing y la suplantación, [Lea esto](https://support.office.com/article/0d882ea5-eedc-4bed-aebc-079ffa1105a3?wt.mc_id=Office_Outlook_com_Alchemy). Para bloquear direcciones de correo electrónico, agregar direcciones o dominios a la lista de remitentes seguros o aplicar otras opciones de correo electrónico no deseado, seleccione **Opciones de correo electrónico no deseado**.
