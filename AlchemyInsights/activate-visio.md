# <a name="activate-visio"></a>Activar Visio

El administrador tiene que asignar una [licencia de Visio](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1800) a un usuario antes de que la aplicación de escritorio de Visio que se incluye con el plan 2 de Visio online pueda instalarse y activarse. (El plan 1 de Visio online no incluye una versión de escritorio de Visio).

1. [Instale Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1800) si todavía no está instalado.
2. Una vez finalizada la instalación, abra la aplicación. En muchos casos, la aplicación se activará automáticamente y el usuario puede empezar a usar Visio inmediatamente.
    Si se le pide que inicie sesión, el usuario debe iniciar sesión con la misma cuenta profesional o educativa a la que se asignó la licencia de Visio.
