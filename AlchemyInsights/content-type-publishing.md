# <a name="content-type-publishing"></a>Publicación de tipo de contenido

Si desea que los tipos de contenido estén disponibles en toda la organización para todos los usuarios, es un buen candidato para publicar en todas las colecciones de sitios. Para publicar un tipo de contenido, vea [publicar un tipo de contenido desde un concentrador de publicación de contenido](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02).
