# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Versión compatible de la herramienta de migración de SharePoint



Actualmente, la herramienta de migración de SharePoint solo admite migraciones de SharePoint 2013. No se admiten las versiones 2010 y 2016 en este momento.
  
Para obtener más información, vea [Introducción a la herramienta de migración de SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
