# <a name="install-and-configure-onedrive"></a>Instalar y configurar OneDrive

El cliente de sincronización de OneDrive está preinstalado con Windows 10. Para instalar OneDrive para Windows 7 o Mac, o para obtener la aplicación móvil de su teléfono o Tablet PC, vaya a [http://onedrive.com/download](http://onedrive.com/download).
  
Para configurar el cliente de sincronización de OneDrive en Windows, consulte:
  
- [Configurar el equipo para sincronizar su para archivos de OneDrive empresarial en Office 365](https://go.microsoft.com/fwlink/?linkid=533375)
    
- [Sincronizar archivos de SharePoint con el nuevo cliente de sincronización de OneDrive](https://go.microsoft.com/fwlink/?linkid=871666)
