# <a name="read-receipts"></a>Confirmaciones de lectura

No puede solicitar una confirmación de lectura en Outlook.com, pero puede solicitar confirmaciones de lectura para los mensajes de Outlook.com enviados con Outlook para Windows. Para obtener información sobre cómo hacerlo, vea [Agregar y solicitar confirmaciones de lectura y notificaciones de entrega](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141?wt.mc_id=Office_Outlook_com_Alchemy).
  
Para elegir cómo Outlook.com responde a las solicitudes de confirmación de lectura, vaya a [configuración de confirmación de lectura](https://outlook.live.com/mail/options/mail/handling/readReceipts).
