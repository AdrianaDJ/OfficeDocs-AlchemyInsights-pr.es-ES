# <a name="data-location"></a>Ubicación de datos

Mantenemos varias copias de los datos en los centros de datos para obtener redundancia. [Vea dónde se encuentran los datos](https://office.com/datamaps). Si ampliamos la ubicación de los datos a un nuevo país o región, le ofrecemos un aviso de avance de un mes.
  
Para obtener más información acerca de la seguridad, privacidad y cumplimiento de Office 365, consulte el [centro de confianza de office 365](https://products.office.com/business/office-365-trust-center-welcome).
