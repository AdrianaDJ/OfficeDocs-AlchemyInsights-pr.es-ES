# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Convertir un buzón de usuario en un buzón de sala o equipamiento

Puede convertir el buzón de un usuario en un buzón de sala o de equipamiento mediante Exchange Online PowerShell. Para obtener más información, vea [convertir un buzón](https://go.microsoft.com/fwlink/p/?LinkId=832875) .
