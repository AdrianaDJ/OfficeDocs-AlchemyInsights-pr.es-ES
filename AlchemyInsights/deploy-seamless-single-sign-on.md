# <a name="enable-seamless-sso"></a>Habilitar SSO sin problemas

Azure Active Directory (Azure AD) el inicio de sesión único sin interrupciones (SSO transparente) inicia sesión de forma automática en los usuarios cuando están en sus escritorios corporativos conectados a la red corporativa.
  
[Siga los pasos de este artículo para habilitar el SSO sin problemas](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
