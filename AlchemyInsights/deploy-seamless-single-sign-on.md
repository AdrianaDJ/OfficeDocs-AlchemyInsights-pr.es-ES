# <a name="enable-seamless-sso"></a>Habilitar SSO perfecta

Azure Active Directory (AD Azure) perfecta Single Sign-On (SSO perfecta) inicia sesión automáticamente a los usuarios cuando se encuentran en sus escritorios corporativos que están conectados a la red corporativa.
  
[Siga los pasos descritos en este artículo para habilitar SSO perfecta](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
