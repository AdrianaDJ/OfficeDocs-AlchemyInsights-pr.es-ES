"Sistema operativo no compatible"

# <a name="solutions-for-unsupported-operating-system"></a>Soluciones para el "Sistema operativo no compatible"

Cuando intenta instalar Office 365/2016, es posible que vea el siguiente mensaje de error si su dispositivo no cumple los [requisitos del sistema](https://products.office.com/office-system-requirements).
  
 **"Sistema operativo no compatible"**
  
Office 365/2016 requiere:
  
PC: 10 de Windows, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 o Windows Server 2008 R2
  
O BIEN
  
Mac OS X 10.10
  
Si obtuvo este mensaje al intentar instalar Office en un dispositivo móvil o un Chromebook, vea [Configurar aplicaciones de Office y correo electrónico en un dispositivo móvil](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
