# <a name="solutions-for-unsupported-operating-system"></a>Soluciones para el "Sistema operativo no compatible"



Cuando intenta instalar Office 365, es posible que vea un mensaje de error del **sistema operativo no compatible** si su dispositivo no cumple los [requisitos del sistema](https://products.office.com/office-system-requirements).
  
Office 365 requiere:
  
- PC: 10 de Windows, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 o Windows Server 2008 R2
    
    O BIEN
    
- Mac OS X 10.10
    
Si obtuvo este mensaje al intentar instalar Office en un dispositivo móvil o un Chromebook, vea [cómo instalar y ejecutar Microsoft Office en un Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
