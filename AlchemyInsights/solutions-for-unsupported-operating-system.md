# <a name="solutions-for-unsupported-operating-system"></a>Soluciones para "sistema operativo no admitido"

Al intentar instalar Microsoft 365, es posible que aparezca un mensaje de error de **sistema operativo no admitido** si el dispositivo no cumple los [requisitos del sistema](https://products.office.com/office-system-requirements).
  
Microsoft 365 requiere:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 o Windows Server 2008 R2

    O

- Mac OS X 10.10
