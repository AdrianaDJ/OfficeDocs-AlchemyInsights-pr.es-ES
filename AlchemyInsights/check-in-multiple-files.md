# <a name="check-in-several-files-at-once"></a>Proteger varios archivos a la vez

<p>Proteger y desproteger archivos es una buena forma de realizar un seguimiento y controlar cómo se actualiza el contenido. Si desea realizar cambios en un archivo de un sitio y desea asegurarse de que nadie más pueda editarlo, Desproteja el archivo. Si el archivo se ha desprotegido, puede editarlo en línea o sin conexión y guardarlo&mdash;varias veces, si es necesario.</p> <p>A continuación se muestran algunos vínculos útiles relacionados con esta característica.</p> <ul> <li><a href="https://support.office.com/en-us/article/check-out-check-in-or-discard-changes-to-files-in-a-library-7e2c12a9-a874-4393-9511-1378a700f6de">Cómo desproteger, proteger o descartar cambios en archivos</a></li> <li><a href="https://support.office.com/en-us/article/Top-questions-about-check-out-check-in-and-versions-7E941339-E972-4C7A-A79A-80A1FCF84076">Las principales preguntas sobre desprotección, protección y versiones</a></li> <li><a href="https://support.office.com/en-us/article/check-out-check-in-or-discard-changes-to-files-in-a-library-7e2c12a9-a874-4393-9511-1378a700f6de">Proteger varios archivos a la vez</a></li> <li><a href="https://support.office.com/en-us/article/enable-and-configure-versioning-for-a-list-or-library-1555d642-23ee-446a-990a-bcab618c7a37?ocmsassetID=HA102772148&amp;CTT=3&amp;CorrelationId=52441bb1-a619-4375-89d5-19d28769890f&amp;ui=en-US&amp;rs=en-US&amp;ad=US">Habilitar o deshabilitar el control de versiones y desprotección en una biblioteca</a></li> </ul> <p>&nbsp;</p>
