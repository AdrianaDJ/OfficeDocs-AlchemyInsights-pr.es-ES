# <a name="ignorelegalhold-switch"></a>Conmutador **–IgnoreLegalhold**

El **–IgnoreLegalhold** usado en el siguiente commandlet no está disponible o no es funcional para los objetos de usuario de Exchange Online.

- Disable-mailbox
- Remove-mailbox
- Disable-mailuser
- Remove-mailuser
