# <a name="move-email-to-office-365"></a>Mover correo electrónico a Office 365

Tiene varias opciones para migrar el correo electrónico a Office 365. [Office 365 mail Migration Advisor](https://aka.ms/alchemyinsight-mailmigrationadvisor) le ayuda a elegir el mejor para su entorno y también le guía a través de la migración de correo.
  
Para obtener información sobre las opciones sin iniciar la migración, consulte [decidir una ruta de migración](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Consulte también la página [adopción de Office 365 con FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) para obtener ayuda asistida con la migración, si es elegible.
