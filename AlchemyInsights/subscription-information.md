# <a name="subscription-information"></a>Información de suscripción

Para obtener información acerca de las suscripciones que posee, vaya a la página de [suscripciones](https://go.microsoft.com/fwlink/p/?linkid=842054) o elija **facturación** \> **suscripciones**.
  
Información clave sobre cada suscripción:
  
- Estado de cada uno
    
- Licencias que adquirió
    
- Licencias asignadas
    
- Estado de facturación periódica
    
- equilibrar
    
- Método de pago
    
- Próxima fecha de facturación o expiración
