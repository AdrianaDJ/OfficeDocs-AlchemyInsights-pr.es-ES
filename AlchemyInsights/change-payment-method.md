# <a name="change-payment-method"></a>Cambiar el método de pago

Puede comprobar el método de pago que está configurado de forma que se va a **facturación** \> **suscripciones**. Elija la suscripción que se va a comprobar. El método de pago que está configurado para la suscripción aparece bajo la **forma de pago**. Para actualizar los detalles de la forma de pago existente, elija **actualizar los detalles de pago**.
  
1. En el Centro de administración, vaya a la página [Suscripciones](https://go.microsoft.com/fwlink/p/?linkid=842054) o elija **Facturación** \> **Suscripciones**.
    
2. Elija la suscripción que se va a cambiar el método de pago para, a continuación, elija **Cambiar el método de pago**..
    
3. Siga los pasos en pantalla para completar el cambio.
    
Para obtener instrucciones paso a paso, vea: [cambiar el método de pago de Office 365 para profesionales](https://support.office.com/article/8652f539-3123-4a8f-b9bd-6aa2f0e0372d)
