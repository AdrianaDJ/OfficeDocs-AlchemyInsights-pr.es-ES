# <a name="how-to-prepare-a-mail-merge"></a>Cómo preparar una combinación de correspondencia

Para obtener información acerca de la combinación de correspondencia en diferentes plataformas, siga las instrucciones que se indican en [usar la combinación de correspondencia para el correo electrónico masivo, las letras, las etiquetas y los sobres](https://support.office.com/article/use-mail-merge-for-bulk-email-letters-labels-and-envelopes-f488ed5b-b849-4c11-9cff-932c49474705).
 
Para obtener más información sobre el uso de Excel para una combinación de correspondencia, vea [preparar el origen de datos de Excel para combinar](https://support.office.com/article/prepare-your-excel-data-source-for-a-word-mail-merge-2d802b6b-a3a3-43e5-bb76-2cac7c68673e) correspondencia y [combinar correspondencia con una hoja de cálculo de Excel](https://support.office.com/article/Mail-merge-using-an-Excel-spreadsheet-858c7d7f-5cc0-4ba1-9a7b-0a948fa3d7d3).
