# <a name="unknown-user-appears-in-teams-meetings-or-chats"></a>"Usuario desconocido" aparece en reuniones o chats de Microsoft Teams

Es posible que el usuario se haya eliminado de Microsoft 365. En este caso, pruebe a eliminar el usuario de Microsoft Teams.  

Intente borrar la siguiente memoria caché cuando se cierre Teams y, a continuación, reinicie Teams: `C:\Users\<USER>\AppData\Roaming\Microsoft\Teams`

Para obtener las actualizaciones de este problema, vea el  [usuario eliminado aparece como "usuario desconocido" en Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/removed-user-appears-as-unknown).
