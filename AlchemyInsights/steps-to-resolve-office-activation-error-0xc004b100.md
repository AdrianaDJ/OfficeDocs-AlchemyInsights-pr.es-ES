# <a name="steps-to-resolve-office-activation-error-0xc004b100"></a>Pasos para resolver el error de activación de Office 0xc004b100


Pruebe estas soluciones para corregir el error de activación de Office 0xc004b100:
  
1. Realice una [reparación en línea de Office](https://support.office.com/article/7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b).
    
2. Descargar e instalar el [Ayudante para la recuperación y soporte técnico](https://aka.ms/SARA-OfficeActivation-Alchemy) y ejecute el **Solucionador de problemas de activación**.
