# <a name="activate-microsoft-forms-for-an-organization"></a>Activar Microsoft Forms para una organización

Un administrador puede activar o desactivar Microsoft Forms para su organización. Obtenga información sobre cómo comprobar la activación de la [licencia y el estado de Azure CollabDBService](https://support.office.com/article/Turn-off-or-turn-on-Microsoft-Forms-8dcbf3ab-f2d6-459a-b8be-8d9892132a43) para su espacio empresarial.
