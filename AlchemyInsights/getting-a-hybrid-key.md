# <a name="getting-an-exchange-hybrid-key"></a>Obtención de una clave de híbrida de Exchange

Para conectar la organización local a Exchange Online en Office 365, debe configurar al menos un servidor de Exchange en la organización local. Este servidor se utiliza para crear una implementación híbrida que se conecta a la organización de Exchange local a Exchange Online. Para obtener una clave de híbrido para este servidor, siga los pasos en el [Asistente para la distribución de claves de producto de Exchange híbrida](http://aka.ms/hybridkey).
