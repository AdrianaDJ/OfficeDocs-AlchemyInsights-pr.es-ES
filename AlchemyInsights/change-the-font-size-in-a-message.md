# <a name="change-the-font-size-in-a-message"></a>Cambiar el tamaño de fuente en un mensaje

1. Si ya ha escrito el mensaje, seleccione el texto.
2. Seleccione el icono de  **formato** debajo del mensaje (es el primer icono).
3. Seleccione el icono de  **tamaño de fuente**  encima del mensaje (es el séptimo icono).
4. Seleccione un tamaño.
