# <a name="sync-a-users-profile-picture"></a>Sincronizar la imagen del perfil de un usuario

Sincronización de perfiles de imagen en SharePoint Online requiere una acción de un usuario para iniciar la sincronización de sus fotografías. No hay ninguna sincronización automática de fotos a SharePoint Online. SharePoint Online sólo sincroniza fotos desde Exchange Online, no Active Directory de Azure.
  
Si un usuario nunca ha realizado una acción que tendría una sincronización o han actualizado sus imágenes de perfil y el nuevo uno no aparece en SharePoint Online, el usuario que necesite solicitar que se va a sincronizar. Aquí es cómo puede hacerlo:
  
1. Abra un explorador web en una sesión en privado o Incognito y [vaya a Office.com](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.office.com%2F&amp;data=02%7C01%7C%7Cbad62c504a36446096e108d614dec653%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C636719344369977864&amp;sdata=Km7ZnN8FHSouZbxOiEpQAGIKsK82SHr25uYCh3Gc%2F3s%3D&amp;reserved=0).
  
2. Inicie sesión con su cuenta de trabajo o escuela y haga clic en el icono de OneDrive o SharePoint.
