# <a name="a-user-has-two-mailboxes"></a>Un usuario tiene dos buzones

Entornos híbridos que usan Azure Active Directory Connect (AAD conectar) o DirSync accidentalmente pueden causar un usuario tener dos buzones: uno en la organización local y otra en la nube.
  
Para resolver este problema, consulte [Permanentemente borrar la información buzón anterior](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
