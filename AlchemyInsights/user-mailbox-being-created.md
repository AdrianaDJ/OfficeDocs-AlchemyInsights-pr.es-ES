# <a name="your-users-mailbox-is-being-created"></a>Se crea el buzón del usuario

Puede tardar hasta una hora antes de que un nuevo buzón de correo está listo para usarse.
  
Si el buzón del usuario ha sido "Preparar" para más de una hora, puede intentar eliminar y volver a crear el usuario o usar la opción "háganos le llame" para hablar con un ingeniero de soporte técnico.
