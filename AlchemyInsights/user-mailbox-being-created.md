# <a name="your-users-mailbox-is-being-created"></a>Se está creando el buzón de correo del usuario.

Puede tardar hasta una hora antes de que un nuevo buzón esté listo para usarse.
  
Si el buzón del usuario se ha "preparando" durante más de una hora, puede intentar eliminar y volver a crear el usuario, o bien usar la opción "llámenos" para hablar con un ingeniero de soporte técnico.
