# <a name="yammer---possible-service-issue"></a>Yammer: problema de servicio posible

Es posible que haya un problema temporal en el que los usuarios puedan experimentar errores al obtener acceso a Yammer. Compruebe el [Panel de estado del servicio](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) para ver si esto afecta a la red.
