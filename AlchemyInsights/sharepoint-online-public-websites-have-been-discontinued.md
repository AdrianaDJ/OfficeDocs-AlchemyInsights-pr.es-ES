# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>Se han discontinuado los sitios web públicos de SharePoint Online

Se han discontinuado los sitios web públicos de SharePoint Online.

Si ha creado un sitio web público de SharePoint Online, podrá seguir utilizándolo hasta el 31 de marzo de 2018. Si desea seguir teniendo un sitio web público, debe crear uno nuevo con un nuevo proveedor. Se ha eliminado el contenido de su sitio web público de SharePoint Online.

Para obtener información sobre la descontinuación del sitio web público, vea [detalles sobre el proceso](https://go.microsoft.com/fwlink/?linkid=866980)de descontinuación de características.
