# <a name="access-services"></a>Servicios de Access

Se retira la característica de servicios de Access en SharePoint Online. Ya no se pueden crear nuevas aplicaciones web basadas en Access y bases de datos web y bases de datos de web y las aplicaciones web existentes se cerrará por abril de 2018. Bases de datos de Access Desktop (.accdb) no se ven afectados por este cambio. Se recomienda que Microsoft PowerApps como una alternativa a acceso web de aplicaciones y bases de datos web. 
  
[Obtenga información sobre Microsoft PowerApps](https://powerapps.microsoft.com/)
