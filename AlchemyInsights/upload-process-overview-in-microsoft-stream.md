# <a name="upload-process-overview-in-microsoft-stream"></a>Introducción al proceso de carga en Microsoft Stream

Siempre que cargue un vídeo, Microsoft Stream lo codifica automáticamente en varias velocidades de bits para que el vídeo se pueda transmitir mediante la transmisión por secuencias de bits adaptable. Microsoft Stream optimiza la velocidad de bits y la calidad del vídeo en reproducción en función del ancho de banda disponible del visor.

- [Introducción al proceso de carga en Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
