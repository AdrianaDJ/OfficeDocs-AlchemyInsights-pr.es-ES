# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>¿Obtención de una cuenta había deshabilitada error en Outlook en el web?

Si se produce el error **se ha deshabilitado su cuenta** con `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` al intentar abrir Outlook en la web (anteriormente conocida como Outlook Web App), el Administrador de la organización que haya deshabilitado el acceso a Outlook en la web. El administrador puede corregir este error, siga los pasos descritos en el tema [Habilitar o deshabilitar Outlook Web App para un buzón de correo](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
