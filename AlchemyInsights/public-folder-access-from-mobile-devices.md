# <a name="public-folder-access-from-mobile-devices"></a>Acceso a carpetas públicas desde dispositivos móviles

El acceso a las carpetas públicas no funciona con dispositivos móviles, use OWA (Outlook en la web) o el escritorio de Outlook para Windows o los clientes de Outlook para Mac para acceder a las carpetas públicas.

Si desea tener una funcionalidad similar a la de las carpetas públicas y que el contenido sea accesible en dispositivos móviles, consulte [más información sobre los grupos de 365 de Microsoft](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) para obtener una mejor alternativa.
