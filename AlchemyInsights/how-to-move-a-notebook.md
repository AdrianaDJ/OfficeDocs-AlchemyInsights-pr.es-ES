# <a name="how-to-move-a-onenote-notebook"></a>Cómo mover un bloc de notas de OneNote

* Para mover un bloc de notas de OneNote a OneDrive, vaya a **archivos**  >  **compartidos**  >  **mover blocs de notas** en OneNote 2016. Para obtener más información, vea [mover un bloc de notas de OneNote a OneDrive](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082) .

* Para obtener más información sobre los procedimientos recomendados a la vez que se prepara para mover los blocs de notas, consulte [Best Practices de sincronización de OneNote](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices).

* [Mover un bloc de notas de OneNote para Windows que haya compartido con otros](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4) proporciona más información sobre cómo mover blocs de notas compartidos.
